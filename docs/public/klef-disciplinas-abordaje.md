# Cómo Abordamos las Disciplinas en Klef

Klef opera como un Branded House integrado, donde cada disciplina contribuye al sistema unificado. Abordamos cada área con metodologías centradas en el cliente, pragmatismo y medición. A continuación, el enfoque por disciplina.

## 1. Klef Dev: La Parte Técnica

Abordamos el desarrollo como "ingenieros de sistemas de marca", creando tecnología que respira identidad. No somos un dev shop commodity; integramos diseño y funcionalidad para experiencias coherentes.

### Enfoque

- **Arquitectura Nativa**: HTML5/CSS3/JS sin frameworks efímeros (Web Soberana). Preserva valor acumulado, reduce deuda técnica.
- **Proceso**:
  1. Discovery & Arquitectura (2-3 semanas): Requerimientos, user flows, stack (React/Next.js, Node, GraphQL).
  2. Design System (2-4 semanas): UI en Figma, componentes, responsive.
  3. Desarrollo (6-12 semanas): Frontend/backend, integraciones, testing.
  4. Launch & Soporte (2-3 semanas): Optimización, analytics, training.
- **Diferenciador**: Sistemas escalables con alma; performance nativa (Lighthouse 95+).
- **Resultados**: Websites $15K-70K, apps $50K-200K; soporte post-launch.

## 2. Klef Strategy: La Parte Estratégica

Abordamos la estrategia como "arquitectura de decisión", transformando complejidad en dirección ejecutable. Usamos EC0249 y Design Thinking para mapas que se implementan.

### Enfoque

- **Metodologías**: Design Thinking para empatía/iteración; EC0249 para diagnósticos rigurosos.
- **Proceso**:
  1. Discovery (2-3 semanas): Entrevistas, auditoría, análisis competitivo.
  2. Arquitectura (2-4 semanas): Posicionamiento, messaging, customer journeys.
  3. Activation (1-2 semanas): Presentación, toolkit, training.
- **Diferenciador**: Roadmaps 6-12 meses medibles; no teoría, sí ejecución.
- **Resultados**: Proyectos $8K-50K; % conversión a fases siguientes 80%; NPS alto.

## 3. Klef Brand: La Parte de Branding

Abordamos el branding como "sistemas de identidad que funcionan", creando marcas escalables y coherentes. Expertise colectiva en diseño avanzado.

### Enfoque

- **Sistema Modular**: Klef Pieces (unidades base), Big Patterns (aplicaciones), guidelines.
- **Proceso**:
  1. Strategy (2-3 semanas): Brand audit, positioning, voz.
  2. Naming (1-2 semanas): Opciones, testing.
  3. Identity Design (4-6 semanas): Logo, color, tipo, sistema completo.
  4. Systemization (2-3 semanas): Guidelines, asset library, training.
- **Diferenciador**: Identidades "que respiran"; no logos bonitos, sí sistemas funcionales.
- **Resultados**: Startups $15K-40K, rebrands $40K-100K; portfolio score alto.

## 4. Klef Studio: La Parte de Producción Visual

Abordamos el studio como "producción estratégica con propósito", generando contenido que respira estructura narrativa. Área emergente donde probamos metodologías propias.

### Enfoque

- **Método Klef SVS (Story-Value-Solution)**: Framework conciso para contenido inbound. Estructura piezas como viaje: historia del cliente → valor útil → solución. Integra VSLs, keywords y VPC; educa sin vender agresivamente.
- **Proceso**:
  1. Brief & Concept (1 semana): Objetivo negocio, narrativa visual.
  2. Production (1-2 semanas): Foto/video, diseño gráfico.
  3. Post-Production (1-2 semanas): Edición, optimización.
- **Diferenciador**: Contenido con arquitectura; no relleno, sí significado.
- **Resultados**: Proyectos $3K-60K, retainers $3K-8K; banco de assets organizados.

Cada disciplina se integra: Strategy guía, Brand forma, Studio visibiliza, Dev funcionaliza. Resultado: coherencia total, valor exponencial.
